# Light Pilot v0.2.0

Productization and usability release.

## Highlights

- First-run onboarding for the intended tray-first workflow.
- `Too dim` and `Too bright` feedback buttons.
- Per-display rules for disabling control or applying brightness offsets.
- Battery-aware lighter checks.
- Manual update script for local installs.
- Inno Setup installer script.
- GitHub Actions CI and tag-based release automation.
- README product screenshot.

## Notes

- The app still works fully offline during normal use.
- `scripts\update-local.ps1` is explicit and networked by design.
- Local installer EXE generation requires Inno Setup 6.
