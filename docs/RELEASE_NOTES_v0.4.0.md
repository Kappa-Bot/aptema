# Aptema v0.4.0

Aptema replaces the former Light Pilot product identity while preserving local settings and safety behavior.

Highlights:

- premium tray-first shell, flyout, OSD, onboarding, themes, and Adaptive Horizon brand
- stable multi-display identity and per-display DDC/CI, Windows, or overlay control
- custom profiles, process rules, deterministic automation, confidence, and explainable decisions
- local aggregate learning with feedback Undo; no screenshots, titles, content, cloud, or telemetry
- recovery mode after repeated failed starts and per-display failure isolation
- bounded sanitized diagnostics and explicit privacy-filtered support packages
- SHA-256 update staging, smoke validation, known-good retention, and rollback
- full `Aptema.*` projects, executable, startup registration, shortcut, and installer naming

Unsigned build: SHA-256 protects package integrity but does not authenticate the publisher. SmartScreen may warn until code signing is added.
